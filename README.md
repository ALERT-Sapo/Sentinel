ACLS Clinical Training Tools
UCI Health · ALERT Team
A growing collection of web-based clinical training tools designed to help bedside nurses make faster, more confident decisions during acute cardiac events — before the rapid response or code team arrives.
These tools exist because the gap between knowing ACLS and acting on it in real time is wider than most hospitals measure. A nurse who can identify a rhythm but freezes on the next step costs a patient critical seconds. This project is an attempt to close that gap through accessible, low-barrier education that staff can return to as often as they need.

Tools
ACLS Clinical Decision Walkthrough
A scenario-based reference that walks nurses through the full response pathway for five acute cardiac presentations — pulseless V-fib/V-tach, unstable V-tach with a pulse, stable V-tach with a pulse, pulseless PEA, and asystole.
ACLS Rhythms Quiz
An interactive rhythm recognition quiz built with a simulated ZOLL monitor aesthetic. Presents cardiac rhythms with animated EKG traces and tests identification with immediate feedback and scoring.

Usage
Live at: https://alert-sapo.github.io/Shock-NoShock-Quiz/
To deploy your own copy: download the HTML files and host them on any static web server or GitHub Pages instance. No build process required.
